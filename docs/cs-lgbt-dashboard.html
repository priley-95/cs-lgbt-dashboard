CS LGBT Network - data review

About this tool

This is a prototype tool contains a storyboard-type dashboard. It contains analysis on the 2020 Civil Service People Survey dataset for LGB+ respondents and heterosexual respondents on some key areas.

This work is not an output under national, official or experimental statistics. It does use publicly available data from the Civil Service People Survey, published by the Cabinet Office.

Any blank data indicates data supression under dominance & disclosure rules.

It is possible to zoom in & out of these graphs.

This work has been on a voluntary basis with colleagues who are part of the Civil Service LGBT+ Network data team. They are civil servants. * T Elliot * P Riley * J Smith * J Peart (network coordinator)
